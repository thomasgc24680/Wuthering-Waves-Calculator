함수 정리
1. 캐릭터/무기 선택 페이지
 - 캐릭터/무기 grid 생성
    > switch Tab에 따라 Display되는 것 다름. -> 이미지 불러올 때 switch Tab에 따라 불러오기.
	> 캐릭터/무기 이미지 불러오기
	> 캐릭터/무기 이름 불러오기(data.js)
 - 필터 버튼
    > 버튼 생성
	> 버튼 클릭 시 이벤트
	> 버튼 이미지 불러오기
 - 검색 기능
    > 검색 기능
 - 클릭한 캐릭터/무기 detail 페이지로 type, name 넘기기
    > 클릭한 버튼에 해당하는 Data 넘기기(switch tab type, 클릭한 이미지의 name)
 
2. 캐릭터/무기 상세 페이지
 - 캐릭터 이미지 불러오기
 
 
 
캐릭터 이름 찾는 함수
1. 각 파일에서 캐릭터 이름 가져와서 전역 변수 할당.
2. common 파일 등 파일에 character 이름 찾는 함수 만들어서 해당 함수 불러오기.
